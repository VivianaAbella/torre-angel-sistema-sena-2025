# Sistema de Gestión Torre Ángel

Proyecto desarrollado para Tecnólogo en Análisis y Desarrollo de Software - SENA 2025.

## Descripción
Sistema web en PHP con arquitectura MVC para la gestión administrativa de apartamentos en Torre Ángel. Permite administrar propietarios, pagos de administración, reservas de zonas comunes y generar reportes financieros.

## Tecnologías
- *Backend*: PHP 8+ con patrón MVC
- *Base de Datos*: MySQL 
- *Dependencias*: Composer
- *Frontend*: HTML, CSS, JavaScript

## Contenido del Repositorio
- DOCUMENTACION PROYECTO.pdf: Documentación técnica completa del sistema
- torre_angel.sql: Script de base de datos con estructura y datos
- composer.json / composer.lock: Gestión de dependencias
- index.php: Punto de entrada de la aplicación
